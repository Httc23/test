    前言

    Foreword

    這幾年COVID-19影響世界的層面日漸嚴重促使投資者更加關注企業在ESG的長期價值與治理能力，然而，企業在ESG各面向的指標上需更深刻的去掌握，同時向客戶、員工、投資人、政府等利害關係人描繪企業的全方位價值（Total Value），企業的數位轉型可快速建立與掌握相關的KPI，進行真實且透明的報告揭露，並建立明確的問責制度。

    歐盟在「2030 Digital Compass: the European way for the Digital Decade(European Commission,2021)報告中就指出「在短短一年之內，COVID-19從根本上改變了數位化在社會與經濟的角色與認知」，企業和機關團體必須普遍的應用資訊及通訊技術(Information and Communications Technology)，在職業安全衛生領域數位科技提供創新的潛力與促進工作場所的進步，人類可以思考未來職業安全衛生的潛在挑戰，更好地發揮數位科技的效用。

    數位轉型

    Digital Transformation

    企業開始使用新科技或新技術(智慧協作機器人（collaborative and smart robots, cobots）、大數據、人工智慧與演算法、智慧個人防護具、虛擬實境與增強實境、3D列印)將既有職業安全衛生管理進行流程改造、將數位工具與與工作流程作結合、進一步分析與應用前述數位化工具產出和整理後的數據及擴大使用者的服務提供新價值的轉型。

    鴻才科技安全管理的架構設計上，提供了行動裝置（平板電腦、手機及手錶）、線上平台（巡檢管理、承攬商管理）及管理儀表板(BI、KPI)完整的解決方案、並提供所有的工作者更彈性的工作模式。

    硬體環境建議

    Hardware Environment

    伺服器作業系統：Microsoft Windows Server 2019(含)以上或Ubuntu Linux 20.04 LTS(含)以上

    伺服器資料庫版本：Microsoft SQL Server 2019 standard或MariaDB 10.7.3(含)以上

    Web Server：Microsoft IIS(含)以上或Nginx

    伺服器建議規格：Intel Xeon-Silver 4110(2.1GHz) 2顆/ 64GB DDR4-2666/ 1TB SAS 7.2K 5顆(系統碟200G、本案軟體及Web Server(含儲存照片)300G、資料庫存放區300G)/RAID 5/Gigabit網路介面

    行動裝置配置：支援Android 9(含)以上/iOS 15(含)以上/Windows 10(含)以上、RFID/NFC(無此規格之行動裝置巡檢定位方式僅支援QR Code)、Wi-Fi或4/5G、相機

    RFID定位點：抗金屬/Mifare 13.56 MHz/ ISO 15693/1k byte/IP68/CODESLIX (1024Bit)
